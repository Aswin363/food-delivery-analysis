
# Food Delivery Data Analysis (Innomatics Internship Test)

This repository contains the Jupyter Notebook for the Innomatics Research Labs – Advanced GenAI Internship Entrance Test.

##  Datasets Used
- orders.csv (Transactional Data)
- users.json (User Master Data)
- restaurants.sql (Restaurant Master Data)

##  Tools & Libraries
- Python
- Pandas
- SQLite
- Google Colab

##  Work Done
- Loaded data from CSV, JSON, and SQL formats
- Merged datasets using appropriate keys
- Created final dataset: `final_food_delivery_dataset.csv`
- Performed analysis on:
  - City-wise revenue
  - Cuisine-wise performance
  - Gold vs Regular members
  - Rating-wise revenue
  - Quarterly revenue trends

## Files in this Repository
- Jupyter Notebook (.ipynb) – Complete code and analysis
- final_food_delivery_dataset.csv – Final merged dataset (optional)

##  Objective
To demonstrate data loading, merging, and analysis skills using real-world style datasets as part of the internship entrance test.
